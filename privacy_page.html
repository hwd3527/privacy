개인정보 처리방침

1. 수집하는 개인정보 항목

  • 카메라로 촬영한 명함 이미지

2. 개인정보의 처리 목적

  • 명함 정보 스캔 및 저장
  • 명함 데이터 관리 및 검색
  • 연락처 연동 기능 제공

3. 처리하는 개인정보 항목

  • 명함에 포함된 개인 정보 (이름, 회사명, 전화번호, 이메일 등)
  • 명함 이미지 데이터

4. 개인정보의 처리 및 보유 기간

  • 사용자가 삭제하기 전까지 기기 내 저장
  • 앱 삭제 시 모든 데이터 완전 삭제

5. 개인정보의 제3자 제공

  • 개인정보를 제3자에게 제공하지 않습니다
  • 모든 데이터는 사용자 기기에만 저장됩니다

6. 개인정보 처리 위탁

  • 개인정보 처리 위탁을 하지 않습니다

7. 정보주체의 권리

  • 개인정보 열람, 정정, 삭제 요구권
  • 앱 내에서 직접 수정 및 삭제 가능
